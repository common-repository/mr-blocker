== Description ==

Your website might get a suspicious traffic from certain countries and this plugin is very easy to use , Just to block the unwanted traffic and to keep your website safe , It is a very simple tool to block the annoying traffic , Just check the boxes then block the annoying/suspicious traffic coming from unwanted countries .
== Installation ==

1. Upload the whole folder `mr-blocker` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2. Select the plugin mr blocker from the admin menu in your dashboard
3. Select the countries you want to block
4. Click Block Now
5. You can remove all the blocked countries so that the visitors can access your website .

== Frequently asked questions ==

= A question that someone might have =
How it works ?

The answer is :

We use the ip address of the visitor and then we use an api from www.ipinfo.io so we detect the country of the visitor and then we save the data of the blocked countries in a file Blocked.txt and in case the visitor's country is found in the txt file , then he/she will be redirected to the plugin's website http://mrblocker.extraviews.co/blocked.php 
-The plugin is very simple plugin that you can use without any collision with a code .
-It doesn't even deal with the mysql database on your hosting , It is very safe .
-Once you installed the plugin you will get it in the admin menu in your dashboard .
-In this page you will have the full control to block/unblock the unwanted traffic coming from the unwanted countries .
-After you choose the unwanted countries and then Block them , The visitors from these countries will be redirected to the page of mr Blocker plugin that includes : 
( Sorry , You are blocked from visiting the website . )
